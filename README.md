# JarnoxStockDashboard

## Overview
It's a dashboard to view and analyze stock data with Python backend and interactive frontend.

## Setup Instructions

1. The repository:
git clone [https://github.com/YOUR-USERNAME/JarnoxStockDashboard.git](https://github.com/ManasP99/JarnoxStockDashboard)

2. Go to the project folder:
cd STOCK-DASHBOARD

3. Activate virtual environment:
Windows
.\venv\Scripts\activate
(OR)
Mac/Linux
source venv/bin/activate

4. Install dependencies:
pip install -r requirements.txt

5. Run the backend:
python main.py

6. Open the frontend in a browser:
frontend/index.html

## Project Logic / Insights
- main.py handles backend logic and APIs
- frontend displays interactive stock dashboard
- Concepts/usedTopics.txt contains data references
